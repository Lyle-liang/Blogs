## angular 初始化项目

#### 初始化项目添加路由
- ng new angular --routing

#### 初始化项目添加国际化

```shell
ng add ng-zorro-antd
```

#### 添加蚂蚁组件

```shell
npm install @ngx-translate/core --save
```
